📡 API Endpoints

Health Check
GET /api/v1/health

Check the health status of the API service.

Convert TCP Dump
POST /api/v1/convert

Upload a TCP dump file for conversion to HAR and JMX formats.

Request:

Content-Type: multipart/form-data

file: [PCAP file]
port: 8080 (optional)
host: example.com (optional)
correlation: true (optional, default: true)
parameterization: true (optional, default: true)
threads: 10 (optional, default: 10)
rampup: 10 (optional, default: 10)
loops: 1 (optional, default: 1)

Response:

{
    "job_id": "uuid-string",
    "status": "pending",
    "message": "Conversion job started successfully",
    "links": {
        "status": "/api/v1/status/{job_id}",
        "websocket": "/api/v1/ws/{job_id}"
    }
}

Download Converted File
GET /api/v1/download/{jobId}/{fileType}

Download the converted file. FileType can be 'har' or 'jmx'.

WebSocket Progress Updates
GET /api/v1/ws/{jobId}

Connect via WebSocket to receive real-time progress updates for a conversion job.

Message Format:

{
    "type": "progress",
    "job_id": "uuid-string",
    "status": "processing",
    "progress": 50,
    "message": "Converting HAR to JMX..."
}
List Recent Conversions
GET /api/v1/conversions?limit=20&offset=0

Get a list of recent conversion jobs.

💻 Usage Example

Using cURL
#!/bin/bash

# 1. Upload PCAP file for conversion
response=$(curl -X POST http://localhost:8080/api/v1/convert \
  -F "file=@capture.pcap" \
  -F "correlation=true" \
  -F "parameterization=true")

# 2. Extract job ID
job_id=$(echo $response | jq -r '.job_id')

# 3. Check job status
curl http://localhost:8080/api/v1/status/$job_id

# 4. Download converted files
curl -O http://localhost:8080/api/v1/download/$job_id/har
curl -O http://localhost:8080/api/v1/download/$job_id/jmx
Using JavaScript (WebSocket)
// Connect to WebSocket for real-time updates
const ws = new WebSocket(`ws://localhost:8080/api/v1/ws/${jobId}`);

ws.onmessage = (event) => {
    const update = JSON.parse(event.data);
    console.log(`Progress: ${update.progress}% - ${update.message}`);
    
    if (update.status === 'completed') {
        console.log('Conversion completed!');
        ws.close();
    }
};

ws.onerror = (error) => {
    console.error('WebSocket error:', error);
};

ws.onclose = () => {
    console.log('WebSocket connection closed');
};
